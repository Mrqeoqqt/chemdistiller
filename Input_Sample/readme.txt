0.txt is a minimun input file for chemdistiller software, tested and generated by Dr. Tobias Kind.

0_empty.txt is an empty template for MSP2ChemDistiller.py to generate chemdistiller inputs from MSP files.

example.txt and msp_file_of_example.txt are two relavant files to show the outcome of how MSP2ChemDistiller.py works to convert MSP files to chemdistiller acceptable inputs.
